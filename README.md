# winter-again.nvim

Personal colorscheme. Heavily inspired by the [mountain-theme](https://github.com/mountain-theme/Mountain/).

## Installation

```lua
{
    "winter-again/winter-again.nvim",
    lazy = false,
    priority = 1000,
    config = function()
        require("winter-again").setup({
            -- set config options here
        })
    end
}
```
